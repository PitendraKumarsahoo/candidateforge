# 🚀 Multi-Source Data Transformer

A powerful AI-assisted data transformation platform that collects candidate information from multiple sources, validates it, removes duplicates, resolves conflicts, and generates a single standardized profile.

---

## 📌 Overview

Multi-Source Data Transformer is designed to simplify candidate data management by automatically extracting, merging, validating, and normalizing information from different sources such as resumes, CSV files, APIs, and manual inputs.

The platform minimizes duplicate records, improves data quality, and provides a unified candidate profile for recruiters and HR teams.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/srimanta2005313/Multi-Source-Data-Transformer.git
```

Move into the project

```bash
cd Multi-Source-Data-Transformer
```

Install frontend dependencies

```bash
npm install
```

Install backend dependencies

```bash
pip install -r backend/requirements.txt
```

Start frontend

```bash
npm run dev
```

Start backend

```bash
python backend/main.py
```
